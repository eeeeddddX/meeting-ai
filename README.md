# 🎙️ Meeting AI

Автоматическое распознавание речи и извлечение задач из записей совещаний с помощью локальных нейросетей.

## ✨ Возможности

- ️ Распознавание речи на русском языке (Whisper)
- 🤖 Автоматическое извлечение задач, ответственных и сроков (Qwen2.5)
- 📊 Экспорт результатов в CSV (Excel), JSON и Markdown (Notion)
- 🖥️ Удобный графический интерфейс
- 🔒 Полная приватность — всё работает на вашем компьютере
- 💸 Бесплатно и без ограничений

## 📋 Системные требования

- **ОС:** Windows 10/11 (64-bit)
- **ОЗУ:** минимум 8 ГБ (рекомендуется 16 ГБ)
- **Диск:** ~15 ГБ свободного места
- **Интернет:** нужен только для первоначальной установки
