# 🎙️ Meeting AI

Автоматическое извлечение задач из записей совещаний.

## 🚀 Установка (1 раз)

1. **Запустите `setup.ps1` от имени администратора:**
   - Правой кнопкой → "Выполнить с помощью PowerShell"
   - Дождитесь окончания (5-15 минут)

2. **Запустите приложение:**
   - Дважды кликните на `Meeting AI.lnk` (создастся после установки)
   - Или на `Запустить.bat`

## 📋 Требования

- Windows 10/11
- 8+ ГБ ОЗУ
- 15 ГБ свободного места
- Интернет (для первой установки)

## 🔧 Что устанавливается

- Python 3.12 (если не установлен)
- Ollama + модель Qwen2.5:7b
- Необходимые библиотеки

## 📁 Результат работы

После обработки создаётся папка с файлами:
- `transcript.txt` — расшифровка
- `tasks.csv` — таблица для Excel
- `tasks.md` — для Notion

## ⚙️ Для слабых ПК

В `setup.ps1` замените `qwen2.5:7b` на `qwen2.5:3b`
